# IT Helpdesk Security Lab

Educational IT helpdesk project that will compare a deliberately vulnerable local-only v1 with a SecDevOps-hardened v2.

## Status

The implementation follows [docs/plan-mesa-ayuda-ti.md](docs/plan-mesa-ayuda-ti.md). `v1-inseguro` infrastructure is in progress.

## Security Notice

The future `v1-inseguro` application is intentionally vulnerable and must run only in an isolated local environment. Never deploy it to a public network.
